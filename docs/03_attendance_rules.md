# 勤怠ルール

## 1. 勤務時間の定義
- 所定労働時間：8時間/日
- 休憩：原則1時間（6時間超勤務で必須）
- 勤務時間計算：
  勤務時間 = (退勤 - 出勤) - 休憩時間

## 2. 打刻の制約
- 出勤/退勤/休憩は、同日に対して矛盾しない順序で登録されること
- 二重打刻は拒否する
- 退勤後の休憩開始は拒否する

## 3. 月次締め
- 毎月1日〜末日を集計対象とする
- 勤怠申請の締め期限：翌月3営業日まで
- 締め後の修正は「ADMIN代理修正」となり監査対象

## 4. 遅刻・早退
- 遅刻/早退は勤務時間から差し引かれる
- 遅刻理由の入力を必須とする（出勤が10:00以降の場合）
