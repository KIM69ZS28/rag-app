# 勤怠管理SaaS - 概要

## 1. サービス概要
本サービスは、企業向けの勤怠管理SaaSである。
従業員はWebアプリから出勤/退勤/休憩の打刻を行い、月次勤怠の申請と承認を実施できる。
管理者は従業員マスタ管理、権限管理、各種レポート出力を行える。

## 2. 主要機能
- 打刻（出勤/退勤/休憩開始/休憩終了）
- 月次勤怠の申請/承認
- 有給休暇の申請/承認
- 勤怠集計
- 監査ログ閲覧（管理者のみ）

## 3. 対象ユーザ
- EMPLOYEE（一般社員）
- MANAGER（承認者）
- ADMIN（システム管理者）

## 4. 非機能要件（抜粋）
- 可用性：月間稼働率 99.9%
- 性能：打刻APIはp95で 800ms以内（通常時）
- セキュリティ：認証はCognito、通信はTLS必須
- 監査：勤怠データ変更および権限変更は監査ログを必須
