# トラブルシューティングFAQ（詳細）

## Q1. 打刻ボタンを押しても反映されません
A. 以下を確認してください。
- ネットワーク接続
- Cognitoトークン有効期限切れ
- API Gatewayの5xx/latency増加
- Lambdaエラー増加
必要に応じてRunbook「API 5xx増加」「ログインできない」を参照してください。

## Q2. MANAGERが承認できない
A. 以下の原因が考えられます。
- MANAGERのロールが付与されていない
- 配下メンバーのdepartment_idが紐づいていない
- 申請が締め後（LOCKED）である
管理者（ADMIN）へ問い合わせてください。

## Q3. 勤怠締め後に修正したい
A. 勤怠締め後はEMPLOYEE/MANAGERでは修正できません。
ADMINによる代理修正が必要です。代理修正は監査対象です。

## Q4. 有給残数が合いません
A. 以下を確認してください。
- 休暇申請のステータス（承認済みか）
- ADMINによる直接変更が行われていないか（監査ログ参照）
不明な場合は監査ログを確認してください。

## Q5. エラーコード「AUTH-401」が返ります
A. 認証トークンの不備が疑われます。
- Authorizationヘッダが無い
- Bearer形式ではない
- トークンの期限切れ
再ログイン後も発生する場合は管理者へ連絡してください。
